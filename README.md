## 概要  
筋トレの実績を消費カロリーとそれを基に歩いた距離を計算して、記録するアプリケーションです。  
## 目指した課題解決  
筋トレを漫然と行っていて成果がわからず、モチベーションが上がらない人向け。
毎日行う筋トレの成果を消費カロリー、歩いた距離に変換。
ユーザーの健康を維持するとともに達成感を味わってもらいモチベーションを上げてもらう。
